## 优化方法

### 加载更快

- 减小资源体积：压缩代码，构建工具压缩、gzip
- 减少访问次数：合并代码、服务端渲染、雪碧图，缓存
- 使用更快的网络：CDN

### 渲染更快

- CSS 放 head，JS 放在 body 底部，防止阻塞页面解析
- 尽早执行 JS，用 DOMContentLoad 触发
- 图片懒加载
- 少触发回流和重绘：DOM 查询进行缓存、DOM 操作合并，节流、防抖
